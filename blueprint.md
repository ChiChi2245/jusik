# 기관 투자자 포트폴리오 대시보드

## 1. 개요

본 프로젝트는 국내외 주요 기관 투자자들의 보유 주식 포트폴리오를 시각적으로 확인하고 분석할 수 있는 대시보드 애플리케이션입니다. 사용자는 기관별 포트폴리오를 검색하고, 자산 배분 현황, 보유 종목 비중 등 상세 정보를 확인할 수 있습니다.

## 2. 현재 구현된 기능

### 2.1. 핵심 기능

- **기관 선택 및 필터링:**
    - 기관 목록을 카드 형태로 조회합니다.
    - 국내/해외 기관별로 필터링할 수 있습니다.
    - 검색창을 통해 원하는 기관을 빠르게 찾을 수 있습니다.
- **포트폴리오 대시보드:**
    - 기관 선택 시, 해당 기관의 상세 대시보드로 전환됩니다.
    - **핵심 지표:** 총 평가액, 보유 종목 수, Top 보유 종목을 카드로 요약하여 보여줍니다.
    - **자산 배분 차트:** 기관의 자산 배분 현황을 도넛 차트로 시각화합니다.
    - **보유 주식 비중 차트:** 보유 주식 Top 5의 비중을 파이 차트로 보여줍니다.
    - **보유 종목 테이블:** 전체 보유 종목을 상세 정보(순위, 이름, 티커, 비중, 평가액, 수량)와 함께 테이블로 제공합니다.
- **통합형 종목 상세 분석 패널:**
    - 보유 종목 테이블에서 특정 종목을 클릭하면, 화면 우측에 해당 종목의 상세 정보(시가총액, 샘플 거래내역 및 뉴스)가 담긴 슬라이드 패널이 나타납니다.
- **글로벌 사용자 경험:**
    - **테마 전환:** 깔끔한 UI의 다크 모드와 라이트 모드를 자유롭게 전환할 수 있습니다.
    - **통화 변환:** 모든 금융 데이터를 원화(KRW) 또는 미국 달러(USD) 기준으로 변환하여 볼 수 있습니다.

### 2.2. 디자인 컨셉

- **Clean & Intuitive:** 복잡함을 줄이고 사용자가 기능에 집중할 수 있도록 명확하고 직관적인 디자인을 지향합니다.
- **사용자 피드백 기반:** 현재 UI는 초기 프로토타입이며, 향후 사용자 피드백을 통해 지속적으로 개선될 예정입니다.

## 3. 향후 계획

- **사용자 피드백 수렴:** 현재 구현된 기능과 디자인에 대한 사용자 의견을 바탕으로 개선 방향을 설정합니다.
- **신규 기능 논의:** 안정된 기반 위에서, 사용자가 원하는 새로운 기능(예: 기관 간 비교 분석, '스마트 머니' 흐름 분석 등)의 추가를 신중하게 검토하고 계획합니다.
